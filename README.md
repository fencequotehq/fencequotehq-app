# FenceQuoteHQ

Production-ready Next.js app for FenceQuoteHQ.

## Includes

- Premium landing page
- Fence cost calculator
- Property line / lot size estimator
- Contractor quote score
- Pro contractor estimate mode
- Material takeoff
- Bid proposal PDF generator
- Lead capture form
- Webhook support for Zapier/Make
- Supabase-ready lead API
- Resend-ready email alert API

## Local setup

```bash
npm install
npm run dev
```

## Environment variables

Create `.env.local` from `.env.example`.

## Deploy

Push this code to GitHub, then Vercel will auto-deploy.
